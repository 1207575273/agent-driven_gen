## 六、关键决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 不建独立项目表/部门表 | DISTINCT 查询 | 59 项目 / 7 部门，聚合查询足够 |
| 偏离度在 Python 算 | Service 层 | SQLite 窗口函数支持有限，Python 可测试可调试 |
| 部门筛选用 query param | `?department=应用产品部` | 中文名 URL 编码复杂，统一走 query params |
| 前端下钻用 Modal | 弹窗而非独立页面 | 保持筛选状态和滚动位置 |
| 导入脚本用 pandas | 读 Excel + SQLAlchemy Core 写库 | pandas 处理日期/空值方便，Core 保证表结构一致 |
